artpop.sql, sub.sql and advert.sql are CREATE TABLE scripts for Article Popularity, Subscription and Advertisement facts.

artpop.data.sql, sub.data.sql and advert.data.sql contain SQL INSERT statements with some sample data.

artpop.php, sub.php and advert.php are PHP scripts that generate SQL with INSERT statements. You can modify some of the variables in these files to generate even more data.

sub.canada.csv contains Canadian addresses & cities used by sub.php to generate sample data.
